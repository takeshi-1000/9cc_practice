# 9cc_practice

参考: https://www.sigbus.info/compilerbook

## やること

自分が書いたコンパイラのソースコードを既存のCコンパイラ(cc)を使用して生成されたコンパイラの実行バイナリを使って、
自分が書いたコンパイラのソースコードをアセンブリファイルとして出力すること（セルフホスト）を目標としている
